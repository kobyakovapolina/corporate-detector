🧠 Детектор корпоративного булшита и канцеляризмов

## Goal
Проанализировать 100+ вакансий с hh.ru в двух сферах (IT-аналитики vs Менеджеры по продажам) и ответить на вопрос: **в каких вакансиях больше «пустых» слов и канцеляризмов, и связана ли «вода» с уровнем зарплаты?**

## Stack
- **Python 3.11**
- **Pandas** — данные
- **Requests + BeautifulSoup4** — парсинг
- **Natasha** — токенизация, лемматизация, POS-тегинг (русский язык)
- **Scikit-learn** — TF-IDF, подсчёт метрик
- **Matplotlib + Seaborn** — визуализация
- **WordCloud** — облака слов
