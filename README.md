🧠 Детектор корпоративного булшита и канцеляризмов

## Goal
Проанализировать 1000+ вакансий с hh.ru в двух сферах (IT-аналитики vs Менеджеры по продажам) и ответить на вопрос: **в каких вакансиях больше «пустых» слов и канцеляризмов, и связана ли «вода» с уровнем зарплаты?**

## Stack
- **Python 3.11**
- **Pandas** — данные
- **Requests + BeautifulSoup4** — парсинг
- **Natasha** — токенизация, лемматизация, POS-тегинг (русский язык)
- **Scikit-learn** — TF-IDF, подсчёт метрик
- **Matplotlib + Seaborn** — визуализация
- **WordCloud** — облака слов

## Key Results
1. Вакансии для менеджеров по продажам содержат **на 37% больше прилагательных-оценок** («динамичный», «дружный», «активный») и **в 2.5 раза больше канцеляризмов** («стрессоустойчивость», «коммуникабельность», «обучаемость»).
2. Обратная корреляция (-0.42) между «индексом воды» и указанной зарплатой: **чем пустее текст, тем ниже зарплата**.
3. Топ-3 самых бесполезных слова в вакансиях РФ: *«ответственность»*, *«исполнительность»*, *«клиентоориентированность»*.
